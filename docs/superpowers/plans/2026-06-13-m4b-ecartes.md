# M4b — Écartés (re-sourcer / corbeille) — Implementation Plan

**Goal:** Make the "Écartés" tab live — the other half of the triage loop. Tracks the user
marked Re-sourcer (`status='resourcing'`) or Écarter (`status='trash'`) get a real view with
their reason, a one-click Soulseek-name copy, buy links, and trash management (send to bin /
restore / empty).

**Architecture:** Same augmentation pattern as Revue. Backend list + actions in Rust;
frontend augments the mockup's `renderEcarts` (Tauri only) via a `window.__siftEcarts` hook;
the `ecarts` nav tab is un-hidden from the lean stylesheet. Browser demo untouched.

**Tech stack:** Rust IPC, `navigator.clipboard` for the Soulseek copy, `tauri-plugin-opener`
for buy links (added in slice 3).

---

## Slice 1 — Backend: list + reason + trash management
- `ecartes.rs` (or in filing/ipc): `list_ecartes(conn) -> Vec<EcarteItem>` where
  `EcarteItem { id, path, filename, status, verdict: Option<String>, truncated: bool,
  artist, title }`. `status IN ('resourcing','trash')`, oldest first. artist/title via
  `filing::reconcile_track` (clean name even for moved/trashed files — uses the original
  filename stem). Reason is derived on the front from verdict/truncated/status.
- Reuse `filing::trash_track` for resourcing→bin (file still at original path).
- `restore_track(conn, track_id)`: revert the track's latest live `trash` action (move file
  back from `.sift-trash`, status→`pending`, mark undone). Errors typed.
- `purge_trash(conn, root)`: physically delete `.sift-trash` files for `status='trash'`
  tracks, set `status='purged'`. (Honors `TRASH_PURGE_DAYS` later; M4b purges on demand.)
- IPC in `ipc_filing.rs`: `list_ecartes`, `restore_track`, `purge_trash` (+ emit
  `queue:changed`). Register in `lib.rs`.

## Slice 2 — Frontend: live Écartés view
- `shared/contracts.ts`: `EcarteItem` type. `ipc.ts`: `listEcartes`, `restoreTrack`,
  `purgeTrash` bindings.
- `app.js`: add `if(window.__siftEcarts)window.__siftEcarts();` at the end of `renderEcarts`
  (no-op in browser, like `__siftHome`/`__siftQueue`).
- `sift-live.ts`: un-hide `ecarts` nav; `renderEcartes()` replaces `#content` under Tauri —
  two filtered groups (à re-sourcer / corbeille), per-row reason badge (faux / tronqué),
  raw filename, **Soulseek copy** (`navigator.clipboard`, format `Artiste Titre` single
  space), send-to-bin / restore / empty-bin buttons.

## Slice 3 — Buy links
- Add `tauri-plugin-opener` (Rust dep + init + `opener:allow-open-url` permission).
- `ipc.ts`: `openUrl(url)`. Per-track store links (Beatport · Traxsource · Juno · Bandcamp
  · Amazon · Apple Music) opened in the default browser.

## Out of scope
- Duplicate reason (needs M5 fingerprinting).
- Auto-purge timer (M4b purges on demand; the scheduled purge is later).
